执行 .py 文件

在命令行中输入 Python filename 执行程序，filename 为待执行的代码。

相对路径与绝对路径

cmd 的路径为 C:\Windows\system32，当且仅当待执行代码在当前文件夹下，可以使用相对路径，否则必须使用绝对路径。

举一个例子，假设待执行代码为 program.py，位置在 C:\Windows\system32\，则可以直接输入 python program.py 执行代码；若位置在 E:\program\ 下，则需要输入 python E:\program\program.py 执行代码。

argc

argc 为整型数据，存储命令行参数的个数。

argv

argv 为“指向指针的指针”，即字符串数组，用于存储命令行参数。该参数亦可写作 *argv[]。argc的值即为argv元素的数量。argv[0] 存储的是当前程序的位置。

envp

envp 的类型也是“指向指针的指针”，存放当前程序运行环境的参数.

Python 变量命名规则

  • 可以使用字母或下划线开头，不能使用数字开头
  • 仅数字、字母与下划线可以出现在变量名中
  • Python 关键字不可用于变量名

注意：虽然一些有特殊含义与用途的单词允许被用作变量名（例如 str int），但这样做会引发歧义，因此不建议这样命名变量！
