Python str 类型

  • 不可变对象
  • 可迭代对象 - 字符串的每一个字符可以通过下标访问。

转义字符 \

当转义字符与其他字符连接到一起的时候，允许使用特殊符号，常见的特殊符号：

转义字符 效果
\\ \
\' '
\" "
\n 换行符
\u Unicode 编号（四位十六进制数）
\U Unicode 编号（八位十六进制数）

len() 函数

len() 函数可以用于查找字符串的长度（字符个数）。

字符串比较

字符串进行大小比较时，比较的是首字母 ASCII 码的大小。使用 ord() 查看字符的 ASCII 码，得到整型数据；chr() 将 ASCII 码转化为对应的字符。

random 模块

random 模块中有用于生成随机数的方法。

random.random()

随机生成 0-1 之间的一个小数。

random.randint() 与 random.randrange()

随机生成指定指定范围的整数。区别在于 randint() 中的参数组成闭区间，randrange() 中的参数组成左闭右开区间。

round() 函数

四舍五入函数，用于保留指定位数。

循环结构

Python严格按照缩进执行程序！

while 循环

例题：从 1 开始对自然数累加，当和大于 10000 时停止，求此时加到了哪一个数。

for 循环与 range() 方法

例题：计算 1 到 100 求和。

紧跟 for 的是循环变量，in 后边需要紧跟一个可迭代对象。通俗的讲，可迭代对象就是可以通过下标访问对象中某个元素的对象，常见的可迭代对象有字符串、列表、字典、集合、元组等。文件对象也是可迭代对象，可以通过下标访问文件的某一行；Python 还可以直接通过下标访问 .csv 文件的某一行某一列。

range() 方法有三个参数，完整写法为 range(start, end, step)，分别为开始、结束、步长，三个参数均为整型，start 与 end 组成左闭右开区间且都为整型。
