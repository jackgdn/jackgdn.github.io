运算符 名称

< 小于
== 等于
>= 大于等于
<= 小于等于

Python 支持连续比较的操作，例如 score >= 80 and score < 90 亦可写成 80 <= score < 90

优先级顺序： 括号 > 算术运算符 > 关系运算符 > 逻辑运算符

用于表示优先级的括号只可以是小括号。
