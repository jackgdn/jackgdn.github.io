要求使用 3.x 以上版本的 Python

认识 Python

交互模式

交互式窗口的程序不会存储到硬盘里，如果关闭了交互式窗口，已得到的数据会消失。

脚本模式

允许脚本保存，便于调试与修改。

数据长度

与 C/C++ 不同，Python 中的数据没有长度限制。只需在交互模式中就能够进行大数计算：

Python 无需单独定义数据类型，变量的类型通过赋值得到。

美化代码

  • + - * / = 等前后有空格，例如 pi = 3.14
  • , 前无空格后有空格，例如 print(2 * 2, 2 * 3)
  • 使用有意义的变量名，避免使用 a b c d 这样的变量名

定界符

基本数据类型

  • 整型
  • 浮点型
  • 字符串，使用 " 或 ' 定界符表示
  • 逻辑真值，即布尔类型，有 True 和 False 两个值。Python 是大小写敏感的语言，True 和 False 的首字母必须大写

半角与全角

简单来说，半角字符即占用一个字节（8 个二进制位）的字符，全角字符即占用两个字节（16 个二进制位）的字符。, 为半角字符而 ， 为全角字符。全部的英文字符都是半角字符。

程序如何执行

使用编辑器编写代码 -> 编译器将代码转化为可执行文件 -> 解释器将程序运行结果输出

编译语言与解释语言

将全部代码编译后运行的语言为编译语言，可以单步执行代码的语言为解释语言。

Hello, world!

Python 中所有函数调用等都需要使用括号。

print() 函数用于输出内容，每一个 print() 执行后会默认输出一个换行符，因此如果 print() 中没有内容，则程序会输出一个换行符。
